# Gazi University Campus Mobile App

This project is a campus support application developed collaboratively by students from Gazi University.  
The goal of the application is to provide useful information and tools that help students navigate campus life more easily.

The application includes several features such as:

- University announcements
- Student clubs
- Dining hall menus
- To-do list
- Activities and places to visit in Ankara

## My Contribution

I was responsible for the **announcement and user authentication system** of the application.

This includes developing the **login and registration system** with SQLite database integration and managing user sessions using SharedPreferences.

I also implemented the **Announcements page**, where announcements from the university website are fetched and displayed dynamically. Users can browse announcements and mark important ones using a bookmark system.

Additionally, I developed the **Saved Announcements (Favorites) page**, which allows users to view and manage the announcements they previously bookmarked.

## Technologies Used

- Java (Android Development)
- XML (UI Layouts)
- Android Studio
- SQLite Database
- JSoup (for web scraping)
- SharedPreferences
